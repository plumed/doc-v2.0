33 /*
34 Includes an external input file, similar to "#include" in C preprocessor.
35 
36 Useful to split very large plumed.dat files.
37 
38 \par Examples
39 
40 This input
41 \verbatim
42 c1: COM ATOMS=1-100

44 d: DISTANCE ARG=c1,c2
45 PRINT ARG=d
